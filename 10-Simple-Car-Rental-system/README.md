# Car Rental System 🚗

This is a simple Java console-based Car Rental System.

Users can select a car and enter the number of days to calculate the total rental price.

## Features
- Displays available cars
- User selects a car
- Calculates rental price based on number of days
- Simple and easy user interaction

## Available Cars
1. Swift – ₹2000 per day
2. Innova – ₹4000 per day
3. BMW – ₹8000 per day

## Concepts Used
- Java Basics
- Conditional Statements (if-else)
- User Input (Scanner)
- Arithmetic Operations
