# Simple Quiz Application 🧠

This is a simple Java console-based Quiz Application.

The program asks multiple-choice questions and calculates the user's score based on correct answers.

## Features
- Multiple-choice questions
- Score calculation
- Simple user input using Scanner
- Console-based interaction

## Concepts Used
- Java Basics
- Conditional Statements (if)
- User Input (Scanner)
- Variables and Logic

## Sample Output

===== SIMPLE QUIZ =====

1. Java is a ?
1. Programming Language
2. Operating System
3. Database

Your Score: 3/3

## Purpose

This project helps beginners understand:
- How to create quiz-style programs
- How to handle user input
- How to implement simple logic in Java
